<h1 align="center">HQPorner Downloader Actions</h1> 

### Simple hqporner.com video downloader which runs inside Github Actions using [hqporner_api](https://github.com/EchterAlsFake/hqporner_api)

# Quickstart

### 1. Fork this repository
### 2. Go to Actions tab
### 3. Go to HQPorner Video Downloader workflow
### 4. Put your hqporner.com URL
### 5. Write a folder name (default is "downloads")
### 6. Choose your desired quality
### 7. Choose a filename
### 8. Run the workflow
### 9. After workflow has finished you can download the video inside your downloads folder in the main branch of the repository
## Note: If the video you are downloading is larger than 90MB, the script will split your file into 95MB chunks, you have to download all of them and extract (unrar/unzip) them to get a single file.
